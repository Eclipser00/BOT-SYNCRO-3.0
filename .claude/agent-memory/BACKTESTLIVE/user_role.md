El usuario es un desarrollador de sistemas de trading algorítmico que mantiene dos bots sincronizados:
- Un backtest bot (v7.0) basado en Backtesting.py con Bokeh para visualización
- Un live/paper bot (v2.0) con FakeBroker para desarrollo y MT5 para producción

Trabaja en Python y está familiarizado con la arquitectura de ambos sistemas. Su objetivo es asegurarse de que los resultados del backtest y el live bot sean equivalentes antes de llevar el sistema a producción real en MetaTrader 5.

La estrategia activa es PivotZoneTest con parámetros n1=3, n2=100, n3=5 en timeframes M3/M9.
